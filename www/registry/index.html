HarpJS Theme Registry

A simple registry that lives on Github/Surge.sh and can be updated via an addition to the _data.json file.

There has been an open issue since January 2014 requesting a site directory for HarpJS boilerplates, independent of the harp.io platform. I built out a simple registry that lives on Github/Surge.sh and can be updated via an addition to the _data.json file.
